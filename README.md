# Analisi-Trasferte-Aziendali

📌 Descrizione del progetto
Questo progetto nasce da un’esigenza reale emersa durante la mia esperienza lavorativa all’interno di uno studio professionale.
Inizialmente mi era stato richiesto di creare un semplice file Excel per monitorare i chilometri percorsi durante le trasferte aziendali.
Partendo da questa richiesta, ho deciso di ampliare il progetto sviluppando una soluzione più strutturata per l’analisi dei costi di trasferta aziendali, utilizzando Excel, Power Query e Power BI.
Il progetto è stato realizzato a scopo portfolio e i dati utilizzati sono simulati/fittizi.
________________________________________
🎯 Obiettivo
L’obiettivo del progetto è monitorare:
•	costi delle trasferte aziendali 
•	chilometri percorsi 
•	impatto economico dei dipendenti sulle trasferte 
•	andamento mensile dei costi 
•	distribuzione geografica delle destinazioni 
________________________________________
🛠️ Strumenti utilizzati
•	Microsoft Excel 
•	Power Query 
•	Power BI 
•	DAX 
•	Python (supporto generazione dataset) 
•	GitHub 
________________________________________
⚙️ Processo di sviluppo
1. Creazione dataset
Per simulare un contesto aziendale realistico è stato creato un dataset contenente:
•	dipendenti 
•	clienti 
•	città di destinazione 
•	tipologie di veicoli 
•	consumi carburante 
•	prezzi carburante 
•	trasferte aziendali 
Parte del dataset è stata generata con il supporto di strumenti di Intelligenza Artificiale e Python per simulare dati realistici e poter sviluppare l’analisi.
________________________________________
2. Automazione calcolo distanze
Utilizzando Excel è stato creato un sistema automatico per:
•	associare coordinate geografiche alle città 
•	calcolare la distanza geografica tramite formula di Haversine 
•	stimare la distanza reale applicando un coefficiente correttivo per simulare percorsi stradali reali  
________________________________________
3. Modellazione dati
Il modello è stato strutturato utilizzando:
•	tabelle relazionali 
•	riduzione della ridondanza dei dati 
•	relazioni tra dipendenti, trasferte e veicoli 
________________________________________
4. Analisi e dashboard
In Power BI sono state sviluppate dashboard interattive per analizzare:
•	costo totale trasferte 
•	costo medio trasferta 
•	km percorsi 
•	trend mensile dei costi 
•	incidenza dei costi per dipendente 
•	mappa delle destinazioni aziendali 
________________________________________
📊 Principali KPI
•	Costo totale trasferte 
•	Costo medio trasferta 
•	Totale km percorsi 
•	Numero di dipendenti
